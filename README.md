# 📰 Fake News Detection using Machine Learning

A Machine Learning project that classifies news articles as Real or Fake using Natural Language Processing (NLP).

---

## 🚀 Project Overview

This project uses:
- TF-IDF Vectorization
- Multinomial Naive Bayes
- Scikit-learn Pipeline

to detect whether a news article is real or fake based on its text content.

---

## 📊 Model Performance

- Accuracy: 93.87%
- Balanced precision & recall
- Tested on 8,980 samples

---

## 🛠️ Technologies Used

- Python 3.11
- Pandas
- NumPy
- Scikit-learn

---

## ▶️ How to Run

1. Install dependencies:

pip install pandas scikit-learn

2. Run:

python ff.py

3. Enter a news article when prompted.
